# pos-laundry
Sebelum menjalankan program, install dulu NodeJs lalu buka terminal di vscode dan jalankan "npm install" jika sudah lanjut jalankan "npm run dev".  
Jika menginstall npm tidak berhasil di terminal vscode maka jalankan di cmd dengan hal yang sama, namun perlu di perhatikan untuk masuk terlebih dahulu kedalam folder laundry tersebut.

Jangan lupa untuk melakukan migrate.
